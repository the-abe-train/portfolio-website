Don't wait for Spotify Wrapped for insights into your favourite tracks and artists!

Dynamic Gradient Generator

A tool for web developers to design and export dynamic gradients in CSS!

This tool was inspired by a fellow web developer's existing project. The Dynamic Gradient Generator lets users design beautiful gradients using custom colours, scroll speed, and scroll direction. The output CSS code can be copied directly into your next project, or, with the GitHub API, be published as a Gist.
